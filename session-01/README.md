# 第 1 堂：量化交易是什麼？重建認知

**目標：** 理解量化交易與主觀交易的差異，建立正確期待

---

## 課程內容

### 主觀交易 vs 量化交易：核心差異

| 維度 | 主觀交易 | 量化交易 |
|------|---------|---------|
| 決策方式 | 人為判斷 | 規則/模型驅動 |
| 一致性 | 易受情緒影響 | 紀律性高 |
| 回測能力 | 難以驗證 | 可歷史回測 |
| 擴展性 | 受限於人 | 可多市場多策略 |

### 量化交易的優點

- **紀律性：** 規則寫死，不會「再看一下」
- **一致性：** 同樣條件，反覆執行同樣操作
- **回測驗證：** 用歷史資料快速確認策略是否成立
- **完整紀錄：** 所有交易有數據，可事後檢討

### 量化交易的限制

- **過擬合（Overfitting）：** 策略對歷史資料「硬背」，未來失效
- **市場結構改變：** 過去適用的策略，未來可能不適用
- **技術停損風險：** 系統故障、網路斷線時無法及時反應

### 什麼策略適合懶人散戶

- **Event-driven（事件驅動）：** 除權息、財報公告後的方向
- **均值回歸：** 價格偏離長期均值時進場
- **動能（Momentum）：** 趨勢延續，順勢而為

## 實驗：Excel 均線策略回測

### 任務

用 Excel 跑一套 MA Cross（均線交叉）策略：

1. 抓取感興趣的股票歷史收盤價（20筆以上）
2. 計算 5 日均線（MA5）和 20 日均線（MA20）
3. 規則：
   - MA5 上穿 MA20 → 買進
   - MA5 下穿 MA20 → 賣出
4. 記錄信號发生的日期、價格

### 延伸思考

- 這套策略在什麼股票/時段賺錢？
- 虧錢的時候發生了什麼？

## 課後作業

用 Excel 完成上述 MA Cross 回測（至少 20 筆資料），產出：
- 信號記錄表（日期、價格、動作）
- 粗略計算總報酬

---

## 重點複習

- 量化交易的價值在於**紀律與一致性**，而非「戰勝市場」
- 沒有完美的策略，只有適合當下市場的策略
- **先有想法，再翻譯成規則**——不要用數據湊策略
