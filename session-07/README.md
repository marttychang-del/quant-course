### Fama-French 三因子

由 Eugene Fama 和 Kenneth French 提出，解釋股票報酬的三個來源：

| 因子 | 意義 | 長期溢酬 |
|------|------|---------|
| **市場β（MKT）** | 對整體市場的敏感度 | 正（股票風險溢價） |
| **規模（SMB）** | Small Minus Big — 小型股優於大型股 | 正 |
| **價值（HML）** | High Minus Low — 價值股優於成長股 | 正 |

**實務應用：** 如果你的投資組合長期報酬來自「小型股 + 價值股」，那你事實上是在賭 SMB + HML 因子，而不是选股能力。

### 簡易版 Smart Beta 建構

目標：建立一個**規則驅動**的指數增強組合，不依赖主動選股。

```python
# 概念框架（以台股為例）
# 1. 篩選條件：市值 > 50億、成交量 > 100張/日
# 2. 因子評分：營收成長 40% + 毛利率 30% + ROE 30%
# 3. 權重：因子分數標準化後給予權重

factors = {
    'revenue_growth': 0.40,   # 營收成長權重
    'gross_margin': 0.30,     # 毛利率權重
    'roe': 0.30,              # ROE 權重
}

# 標準化每個因子
for factor in factors:
    df[f'{factor}_z'] = (df[factor] - df[factor].mean()) / df[factor].std()

# 總分
df['score'] = sum(df[f'{f}_z'] * w for f, w in factors.items())

# 取分數最高的前 20 檔
portfolio = df.nlargest(20, 'score')
```

### 避免「因子過擬合」

- **空間驗證：** 因子在不同的股票池（A股、港股、美股）是否都有效？
- **時間驗證：** 滾動視窗測試（rolling window），不要只測完整歷史
- **稀疏原則：** 因子解釋力強的少數變數，勝過一堆微弱的變數
- **單因子測試：** 先確認每個因子單獨有效，再考慮組合

## 重點複習

- **因子投資**的本質是：把投資信念翻譯成可衡量、可驗證的規則
- Fama-French 三因子解釋了市場上一大部分的報酬變異
- Smart Beta 的價值在於**紀律性**——不會因為情緒偏離規則
- **過擬合是量化策略最大的敵人**，單一歷史數據算出來的最優參數在實盤幾乎必敗
