# 量化交易 8 堂課

**對象：** 無程式背景、有基本股票交易知識  
**目標：** 學習為主 + 附帶實作（C級）  
**形式：** 8 堂 × 1.5 小時，理論 + 實驗並行

---

## 課程總覽

| 堂數 | 主題 | 核心產出 |
|------|------|---------|
| 01 | 量化交易是什麼？重建認知 | Excel 均線策略回測 |
| 02 | 金融數據基礎 | Python 抓股價初體驗 |
| 03 | 基礎統計與金融時間序列 | N日波動率計算 |
| 04 | 策略骨架：均線與動能策略 | 完整回測流程 |
| 05 | 風險管理與資金管理 | Kelly % + 部位計算 |
| 06 | 統計套利與事件驅動 | 配對交易潛力股票對 |
| 07 | 因子模型與智慧組合 | 持股邏輯 → 量化因子 |
| 08 | 從回測到實盤 + 系統化框架 | 期末專案提案 |

---

## 期末專案

設計一套自己的完整策略，包含：
1. 進場邏輯
2. 停損規則
3. 資金管理
4. 回測績效
5. 持續觀測指標
